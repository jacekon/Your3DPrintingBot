# Contributing

This project is in early development. If you want to contribute later:

1. Open an issue or discussion on [GitHub](https://github.com/jacekon/Your3DPrintingBot).
2. Fork the repo, create a feature branch, and open a pull request.
3. Keep changes focused and aligned with [PROJECT_SCOPE.md](PROJECT_SCOPE.md).

No formal style guide yet; follow existing code style and add tests where it makes sense.
